# 시간 복잡도

## 시간 복잡도를 나타내는 3가지

- 빅오 표기법: 시간 복잡도가 최대 X시간 걸린다. 즉, 상한이 있다.
- 오메가 표기법: 시간 복잡도가 최소 X시간 걸린다. 즉, 하한이 있다.
- 세타 표기법: 시간 복잡도의 평균 (빅오와 오메가의 교집합)

## 빅오 표기법의 카테고리

다음은 시간 복잡도의 범위이다. (시간 복잡도가 낮은 순이다.)

1. $O(1)$
2. $O(logn)$
3. $O(n)$
4. $O(nlogn)$
5. $O(n^2)$
6. $O(n^3)$
7. $O(2^n)$
8. $O(n!)$

## 마스터 정리 - 카테고리로 쉽게 구할 수 없는 시간 복잡도

공식을 이용해 시간 복잡도를 구할 수 있다.

$T(n) = aT(n/b) + f(n)$ 과 같은 모양을 가진 점화식은 마스터 정리 공식을 이용해 카테고리의 시간 복잡도로 변환할 수 있다.

### 마스터 정리 공식

- $g(n) > f(n)$ : 시간 복잡도는 $g(n)$이  결정한다.
- $g(n) < f(n)$ : 시간 복잡도는 $f(n)$ 이 수행시간을 결정한다.
- $g(n) = f(n)$ : 시간 복잡도는 $g(n) * logn$ 이 수행시간을 결정한다.
- $a$, $b$, $g(n)$, $f(n)$ 이 사용된다.

### 예제

$T(n) = 9T(n/3) + n$ 일 때 시간 복잡도는?

- $a$ 는 $T$의 앞에 있는 `9`이다.
- $b$ 는  $T(n/3)$의 `3`이다. (n은 빼고 상수의 분모 분자를 역전한 값인듯)
- $f(n)$은 점화식의 맨 마지막 요소인 $`n`$이다.
- $g(n)$ = $`n^{log_ba}`$
    - $n^{log_ba}$ 에 대입 = $`n^{log_39}`$ = $`n^2`$
- $f(n)$인 $n$과 $g(n)$인 $n^2$  를 비교해보면 $`n^2`$ 가 더 크다는 것을 알 수 있음
- $\theta(g(n))$ = $`\theta(n^2)`$
